* Firebase Realtime Database
  * firebase.js 에 `databaseURL` 추가
    ```js
    const firebaseConfig = {
      apiKey: "apiKey",
      // ....
      databaseURL: "https://databaseName.firebaseio.com",
      // For example, https://your-database-123.europe-west1.firebasedatabase.app
    };
    ```
